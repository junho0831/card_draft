# Card Draft 게임 기획

이 문서는 현재 구현의 게임 규칙과 제품 방향만 기록한다. 세부 UI 규칙은 `docs/ui-responsive.md`, 카드 데이터 작성법은 `docs/card-authoring.md`를 따른다.

## 제품 목표

`Card Draft`는 5-10분 안에 빌드를 완성하고 강한 연계 턴을 만드는 싱글플레이 로그라이크 덱빌딩 RPG다.

- 추천 행동만 따라도 첫 전투를 진행할 수 있어야 한다.
- 같은 빌드 태그를 모으면 플레이 방식이 실제로 달라져야 한다.
- 보상은 카드 설명보다 `왜 지금 좋은지`를 먼저 보여준다.
- 강한 연출은 연계, 처치, 강타, 승리에 집중한다.
- PvP, 시즌, 카드팩, 온라인 저장은 범위에 포함하지 않는다.

## 런 구조

기본 Act는 다음 5개 레이어로 구성한다.

```text
battle -> event/shop -> battle -> rest/shop -> boss
```

새 런은 세력 선택, 맵 진행, 전투와 보상, 보스, 결과 순으로 진행한다. 현재 맵은 레이어별 선택 구조이며 엘리트는 기본 흐름에서 제외한다.

기본 시작 상태:

- 영웅 체력 26
- 골드 85
- 시작 덱 10장
- 선택 세력 카드 9장 + 공용 카드 1장

## 세력

| 세력 | 시작 방향 | 전투당 1회 필살기 |
| --- | --- | --- |
| 인간 | 소환·버프 | `왕국의 집결`: 1/2 근위대 소환, 모든 아군 공격 +1 |
| 엘프 | 드로우·소환 | `바람의 순환`: 카드 2장 드로우, 현재 마나 +2 |
| 언데드 | 사망·소환 | `죽음의 계약`: 아군 희생, 적 영웅 피해 3, 1/1 해골 소환 |

필살기는 마나와 카드를 소비하지 않고 카드 사용 수와 연계 카운터에도 포함되지 않는다. 필살기로 발생한 소환과 사망은 유물·빌드 훅을 정상 발동한다. 언데드는 희생할 아군이 없으면 사용할 수 없다.

내부 데이터의 `중립` 카드는 화면에서 `공용 · 모든 세력`으로 표시한다. 인간은 금색, 엘프는 녹색, 언데드는 보라색, 정령은 청록색, 공용은 강철 회색 프레임을 사용한다.

## 전투 규칙

- 적 영웅 체력을 0으로 만들면 승리하고 플레이어 체력이 0이면 패배한다.
- 양쪽 필드는 5칸이며 유닛끼리 전투하면 동시에 피해를 준다.
- 마나는 1에서 시작해 턴마다 최대치가 1씩 증가하며 상한은 10이다.
- 시작 손패와 턴 시작 드로우는 5장, 손패 제한은 10장이다.
- 턴 종료 시 남은 손패를 버리고 덱이 비면 버림 더미를 섞는다.
- 덱과 버림 더미가 모두 비면 드로우 대신 영웅이 피해 1을 받는다.
- 저주와 의식은 턴 시작에 각각 피해와 회복을 처리하는 1회성 스택이다.

전투 UI는 `추천 카드/공격 -> 남은 행동 -> 턴 종료` 순서로 안내한다. 적 행동은 `유닛 공격`, `영웅 공격 위험`, `소환 준비`처럼 짧게 예고한다.

## 빌드와 연계

카드 1장은 태그 점수 1, 유물 1개는 2를 제공한다. 점수 5 이상이면 빌드가 활성화된다.

| 태그 | 활성 효과 | 플레이 감각 |
| --- | --- | --- |
| `fire` | 화염 피해 +2 | 한 턴 폭딜 |
| `draw` | 턴 시작 추가 드로우 1 | 긴 콤보 턴 |
| `death` | 아군 사망 시 적 영웅 피해 1 | 희생 이득 |
| `buff` | 소환 유닛 체력 +1 | 필드 성장 |
| `low_hp` | 체력 50% 이하에서 유닛 공격 피해 +1 | 위기 역전 |
| `summon` | 전투 시작 1/1 토큰 소환 | 물량 압박 |

같은 활성 태그 카드를 한 턴에 연속 사용하면 2연계부터 태그별 보너스가 발동한다. 다른 태그를 사용하거나 턴이 끝나면 흐름이 초기화된다. 한 전투의 첫 3연계는 빌드별 피니시를 한 번 발동하며 사용 상태는 전투 스냅샷에 저장한다.

대표 앵커 유물은 각 빌드의 리듬을 강화한다.

- 화염: `burning_heart`
- 드로우: `world_tree_leaf`
- 버프: `knight_banner`
- 사망: `necromancer_ring`
- 저체력: `blood_chalice`
- 소환: `war_drum`

## 카드와 보상

카드 타입은 `unit`, `spell`, `equipment`다. 유닛은 필드에 남고, 주문은 즉시 처리되며, 장비는 아군에게 지속 효과와 배지를 부여한다.

일반 전투 보상:

- 골드 15-25
- 카드 3장 중 1장
- 도전 달성 시 골드 10 추가

보스 보상:

- 골드 50
- 카드와 유물
- 최대 체력 +5
- 도전 달성 시 골드 15 추가

카드 보상은 다음 순서로 후보를 배치한다.

1. 현재 세력 + 최고 빌드 태그
2. 공용 카드, 없으면 같은 세력 카드
3. 전체 카드 풀

추천 이유는 `바로 활성`, `연계 카드 확보`, `활성까지 N`, `즉시 전투 도움`처럼 짧게 표시한다.

## 노드와 메타

- 이벤트: 체력·골드 교환, 카드/유물 획득, 강화, 제거
- 상점: 카드 35골드, 유물 110골드, 회복 45골드, 카드 제거 45골드부터 시작
- 휴식: 최대 체력의 30% 회복 또는 카드 1장 강화
- 메타 진행: 영혼석, 시작 체력, 시작 골드, 두 번째 기회

보스는 일반 적과 구분되는 한 줄 기믹을 가진다.

- 국경 수호자: 매 적 턴 선봉 공격 +1
- 언데드 왕: 3턴마다 해골 지원
- 강령술사 군주: 매 적 턴 저주 +1

## 저장

- `user://run_state.json`: 현재 Act, 노드, 덱, 유물, 체력, 골드, 전투/보상/상점/이벤트 상태
- `user://meta_profile.json`: 메타 강화, 설정, 컬렉션

구버전 저장에 세력이나 필살기 상태가 없으면 인간, 필살기 미사용으로 복원한다.
