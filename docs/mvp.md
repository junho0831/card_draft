# Card Draft MVP

## 목표

`Card Draft`의 현재 목표는 온라인 TCG 검증이 아니라, `빌드 완성형 싱글플레이 로그라이크 덱빌딩 게임`의 코어 루프를 Godot 안에서 완성하는 것이다.

핵심은 카드 수집보다 빌드 완성이다.

- 사망 빌드
- 드로우 빌드
- 버프 빌드
- 화염 피해 빌드
- 저체력 역전 빌드

## 현재 범위

- 메인 메뉴에서 `새 런 시작`, `이어하기`, `메타 강화`, `카드 도감`
- 시작 종족 선택 없음
- Act 2개
  - 국경지대
  - 죽음의 성
- 노드 흐름
  - 일반전투
  - 일반전투
  - 이벤트
  - 일반전투
  - 상점
  - 엘리트
  - 휴식
  - 보스
- 전투 승리 후 카드 3장 제시, 1장 선택
- 런 전용 상점, 이벤트, 휴식
- 보스 승리 시 최대 체력 +5
- 영혼석과 메타 업그레이드
- 런 저장/이어하기

## 콘텐츠

- 스타터 카드 4종
- 런 카드 풀 20장
- 유물 15개
- 이벤트 5개
- 일반 적 6종
- 엘리트 2종
- 보스 2종

## 전투 규칙

- 영웅 체력 0이 되면 패배
- 적 영웅 체력을 0으로 만들면 승리
- 마나는 1부터 시작, 턴마다 +1, 최대 10
- 필드는 플레이어/적 각각 5칸
- 유닛 전투는 동시 피해
